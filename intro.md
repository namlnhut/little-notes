# Welcome to Little Notes

This is a collection of notes covering fundamental topics in data science and scientific computing.

## Contents

This book is organized into four main sections:

### Machine Learning
Explore supervised learning, unsupervised learning, deep learning, and model evaluation techniques.

### Elementary Scientific Computing
Learn the basics of numerical computing with NumPy, data manipulation, visualization, and numerical methods.

### Mathematics
Cover essential mathematical foundations including linear algebra, calculus, optimization, and probability theory.

### Statistics
Understand statistical concepts from descriptive statistics to hypothesis testing and regression analysis.

## Getting Started

Navigate through the sections using the sidebar on the left. Each section contains multiple chapters with theory, examples, and practical implementations.
